# Shopez Frontend

## Project Overview

This project contains the frontend for the Shopez e-commerce platform.
It uses the shopez-api backend to create a full-stack (basic) e-commerce application.
This project is part of my learning pathway on Codecademy.com - Full Stack Engineer path.

## Technologies Used

- React
- React Router
- Vite
- MUI

## Features
### Completed (with API integration)
- User can view all available products, including image, description, price
- User can view a selected product with additional product details
- User can add a product to a cart
- User can remove a product from their cart
- User can checkout their cart (No real payment integration etc. as this is just a portfolio project) 
### In Progress / Future
- User login and authorisation using Username / Password
- User login and authorisation using third party (Google, Facebook)
